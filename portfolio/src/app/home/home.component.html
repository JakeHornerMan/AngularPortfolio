Roadster Run

Technologies

  • C#
  • Unity
  • Blender

Roadster Run was my final year project during my time at NCI. It is a mobile phone game with motion controls and Augmented reality functionality.

Gameplay

The gameplay is a car that moves on a procedurally generated road, where the enemy will come to attack the player.

Customization

The player can also customize their player car, which is reflected in the main game and AR functionality.

Augmented Reality

The AR functionality allows players to take images with their customized vehicles.

All art assets for the game were created in Blender.

Blobber

Technologies

  • C#
  • Unity
  • Adobe Photoshop

Blobber is a competitive platformer where players control an escaped alien jumping on scientists for points. The Player will need to dodge fireballs from the hazmat enemy that is flying in the sky to conserve health. ‘Classic’ mode is a time trial mode where the player completes for points, and ‘Floor is Lava’ mode will damage the player when he touches the floor. There is a leaderboard integrated with ‘Google Play Games’, which allows players to complete on each mode, and an opt-in Ad that allows the player to gain 1 extra hp.

Shy Chomper

Technologies

  • C#
  • Unity
  • Adobe Photoshop
  • Unity WebGL

This is a game created in 7 days for the 'Weekly Game Jam' 203, theme 'Eat The Enemy'. This game is created by the Team of Joshua Hall and Jake Horner. This is an action stealth puzzle game.

The Players main goal is to make it to the finish line, and avoids enemies ‘Blobbers’ and ‘Floating Eyes’ from spotting them. It seems straight foreward, but the catch is the Player can only cross tiles marked by a specific color, after consuming a ‘Blobbler’ type enemy of the corresponding color.

We created 5 levels to showcase the toll we built for this game inspired by mazes, board games, and top-down stealth games. Please feel free to follow the link below to give the levels a try!

This was a team submission to GMTK Game Jam 2024, which placed 263rd out of nearly 7000 submitted games. The theme of the Game Jam was “roles reversed”, so we decided to do a reverse on the game ‘Crossy Road’. In ‘Crossy Road’, the player's aim is to control a chicken to get across a road, in our demo ‘Scrambled Eggs’ the player will place cars to stop waves of chicken from crossing the Road. We expanded on this idea with different car types and different chicken types.
